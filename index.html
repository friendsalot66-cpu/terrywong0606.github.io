About Multi-API Chat

A versatile, static web application designed for power users who need access to multiple advanced LLMs like Perplexity (Sonar) and Moonshot (Kimi) in a single, unified interface.

Getting Started

Since this is a client-side application, your API keys are stored locally in your browser and are never sent to any intermediate server.

üîë API Keys Required

To use this application, you need to obtain API keys from the respective providers:

  • Perplexity: Get API Key
  • Moonshot AI: Get API Key

Click the Key icon in the sidebar to enter your credentials.
